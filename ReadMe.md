# ukagakaExe_googleCalendar
伺か/SSPからグーグルカレンダーにアクセスして今日の予定や来週の予定を取得するSAORI-BASICです。
設定が少し面倒で、下記が必要になります。

- Google Cloudのプロジェクト/GoogleCalendarが有効化されていること。
- 同プロジェクトから認証用のjsonファイル
- GoogleCalendar側でプロジェクトと共有したgmail
